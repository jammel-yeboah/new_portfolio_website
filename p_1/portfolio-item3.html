The Correalation Between Firearm Ownership and Gun Violence

Will reducing the number of people who own firearms actaully reduce gun violence in America? This is a research project I am currently working on, where I use gun ownership data sets, census data, and deaths by gun shot per state to answer this question.

The primary data set I would be using estimates gun ownership by state using several different metrics. I would be using the R programming language, along with Python to analyse the data.
